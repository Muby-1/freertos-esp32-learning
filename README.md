This repository is a beginner-friendly guide to learning FreeRTOS using
ESP32 and ESP-IDF. The goal is to understand core FreeRTOS concepts
through simple, practical examples.

## Who is this for?
- Beginners in FreeRTOS
- Embedded developers using ESP32
- Students and professionals preparing for embedded interviews

## Requirements
- ESP32 / ESP32-S3 development board
- ESP-IDF v5.x
- Basic knowledge of C programming

## Learning Structure
This repository is organized in a step-by-step manner:

1. FreeRTOS Basics
2. Task Management
3. Inter-task Communication
4. Timers
5. Event Groups
6. ESP32-specific FreeRTOS features

Each folder contains:
- One concept per example
- Simple and readable code
- A README explaining the concept
